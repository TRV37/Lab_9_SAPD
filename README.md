Выполнил задание по варианту (условие ниже) на тему "Бинарные деревья".

Задача: Генеалогическое дерево человека строится следующим образом. 
Корень дерева соответствует данному человеку. 
Сыновья каждой вершины описывают родителей соответствующего вершине человека. 
Имеется два генеалогических дерева. 
Требуется определить, являются ли эти люди:
двоюродными братьями или сестрами;
сводными братьями или сестрами.
